# 📝 Task Tracker

A full-stack **Task Tracker** web application built using the **MERN Stack** (MongoDB, Express.js, React.js, Node.js). It allows users to efficiently manage their daily tasks with features like creating, updating, deleting, searching, filtering, sorting, and tracking task status.

## 🚀 Live Demo

- **Frontend:** https://task-tracker-kur5.onrender.com
- **Backend API:** https://task-tracker-backend-8d6c.onrender.com

---

## ✨ Features

- ➕ Create new tasks
- ✏️ Update existing tasks using a popup modal
- 🗑️ Delete tasks with a confirmation dialog
- 🔄 Change task status (Pending → In Progress → Completed)
- 🔍 Search tasks by title
- 🎯 Filter tasks by status and priority
- ↕️ Sort tasks by:
  - Newest First
  - Oldest First
  - Due Date
  - Priority (High → Low / Low → High)
  - Title (A → Z / Z → A)
- 📅 Due date management
- 📱 Responsive user interface
- ☁️ Cloud database using MongoDB Atlas
- 🌐 Deployed on Render
